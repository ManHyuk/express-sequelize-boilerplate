## Install And Use

---

1. Start by cloning this repository

```
# HTTPS
$ git clone https://github.com/ManHyuk/express-sequelize-boilerplate.git

$ cd express-sequelize-boilerplate

$ rm -rf .git
```

2. Install module

```
$ npm install
```

3. START

```
$ npm start
```

4. TEST

```
$ npm test
```
